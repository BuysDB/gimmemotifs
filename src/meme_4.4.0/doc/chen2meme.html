chen2meme

Usage: chen2meme [options]

Description:

Read a CHEN matrix file and convert to MEME format. You may concatenate files for more than one motif.

Input:

Reads standard input.

Concatenated CHEN files describe one or more motifs. Each motif has a header line and a position-weight matrix.

A log-odds matrix and a probability matrix are output for each motif. The probability matrix is computed using pseudo-counts consisting of the background frequency (see -bg, below) multiplied by the total pseudocounts (see -pseudo, below). The log-odds matrix uses the background frequencies in the denominator and is log base 2.

Options:

  • -numseqs - assume frequencies based on this many sequences; default: 20
  • -skip <ID> - skip this ID (may be repeated)
  • -bg - file with background frequencies in MEME background file format; default: uniform
  • -pseudo <A> - add <A> times background frequency to each count when computing letter frequencies default: 0
  • -logodds - print log-odds matrix, too; default: print frequency matrix only
  • -h - print usage message

Output:

Writes MEME format to standard output.
