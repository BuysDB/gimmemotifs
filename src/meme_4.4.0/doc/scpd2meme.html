scpd2meme

Usage: scpd2meme [options] <matrix file>

Description:

Convert an SCPD matrix file into a MEME version 3 formatted file suitable for use with MAST and other MEME Suite programs.

Input:

An SCPD matrix file describes one ore more motifs in terms of letter frequencies.

A log-odds matrix and a probability matrix are output for each motif in the file. The probability matrix is computed using pseudo-counts consisting of the background frequency (see -bg, below) multiplied by the total pseudocounts (see -pseudo, below). The log-odds matrix uses the background frequencies in the denominator and is log base 2.

Options:

  • -skip <scpd ID> skip this ID (may be repeated)
  • -numbers use numbers instead of strings as motif IDs
  • -bg <background file> file with background frequencies in MEME background file format; default: uniform frequencies
  • -pseudo < total pseudocounts> add <total pseudocounts> times each background frequency; default: 0

Output:

Writes MEME format to standard output.
